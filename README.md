# Chatbot customer services using RASA framework 
 
### Author: @ducnguyen2133
###Date: 2024/11/15
###Build basic chatbot 
###Python 3.10.0
###Rasa 3.6.20
###Following cmd:
1: ` python -m venv venv`
2: `.\venv\Scripts\activate`
activate venv and install rasa 
3: `pip install rasa`
4: `rasa init`
Now, can run chatbot.
